# Mission Built

**A field guide for building things that matter.**

An open-source book by [Mike Nichols](https://www.linkedin.com/in/hmikenichols), now in its second edition. Lessons from twenty-plus years between two unforgiving rooms: the gym and the boardroom.

The book is free. The website that hosts it is open source. Both live here.

## Read it

Live at **[missionbuilt.io](https://missionbuilt.io)**. Thirteen chapters, three parts, prologue and conclusion. About 3 hours and 37 minutes if you read it straight through.

Or take it offline:

- [Download the PDF](https://missionbuilt.io/download)
- [Download the markdown manuscript](https://missionbuilt.io/downloads/mission-built.md)

## Repos here

- **[missionbuilt-book](https://github.com/missionbuilt/missionbuilt-book)** — the book content. Markdown manuscript, chapter sources, edits.
- **[missionbuilt-sitedesign](https://github.com/missionbuilt/missionbuilt-sitedesign)** — the Astro site that renders it. Forkable. The Iron Log design system documented in the repo.
